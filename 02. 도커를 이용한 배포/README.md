##  Docker 이미지를 빌드하는 2가지 방법

- 커밋 - 컨테이너의 현재 상태를 이미지로 저장
- 빌드 - Dockerfile(도커 이미지 생성관련 스크립트 파일)을 통해 이미지를 생성하여 저장
